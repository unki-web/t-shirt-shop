### 1.3 Оценка дизайна

**Что реализовано хорошо:**
- Чёткая дизайн-система с CSS-переменными
- Последовательная типографика (Playfair Display + Inter)
- Золотой акцент `#C8A96E` как единственный цветовой акцент
- Hover-анимации на карточках товаров
- Минималистичный стиль без лишних элементов

**Что можно улучшить:**
- Добавить реальные фотографии вместо SVG-иконок футболок
- Усилить визуальную иерархию на мобильных устройствах
- Добавить skeleton-загрузку для товаров

---

### 1.4 Оценка качества кода

**Сильные стороны:**
- Использован Vue 3 Composition API (`setup()`, `ref()`, `computed()`)
- Реактивное состояние для корзины, фильтров, шагов checkout
- CSS-переменные для дизайн-системы
- Чистая структура `setup()` с разделением данных и методов

**Области для улучшения:**
- Разбить на отдельные компоненты (сейчас всё в одном файле)
- Добавить Pinia для управления состоянием
- Добавить Vue Router для настоящего роутинга
