## 1. Что продаём

**MONOCHROME** — интернет-магазин премиальных футболок в минималистичном стиле.

**Целевая аудитория:** люди 20–35 лет, ценящие качество, лаконичный дизайн и осознанное потребление.

**Ключевые характеристики продукта:**
- Футболки из 100% органического хлопка
- Лимитированные капсульные коллекции
- Нейтральная цветовая палитра (белый, чёрный, серый, бежевый)
- Цена: 2 500 – 6 000 ₽

## 3. Техническое задание

### Стек технологий

| Слой | Технология |
|------|-----------|
| Frontend | Vue 3 (Composition API) |
| Стилизация | CSS Variables + Tailwind CSS |
| Роутинг | Vue Router 4 |
| Стейт | Pinia |
| Запросы | Axios / Fetch API |
| Сборка | Vite |
| Хостинг | Vercel / Netlify |

### 3.1 User Flow

```
Главная страница
    │
    ├─► Каталог (фильтры: цвет, размер, цена)
    │       │
    │       └─► Карточка товара
    │               │
    │               ├─► Выбор размера → Добавить в корзину
    │               └─► Избранное (wishlist)
    │
    ├─► Корзина
    │       │
    │       └─► Оформление заказа
    │               │
    │               ├─► Данные покупателя
    │               ├─► Доставка
    │               ├─► Оплата
    │               └─► Подтверждение ✓
    │
    └─► О бренде / Контакты
```

### 3.2 Дизайн-проект

**Цветовая палитра:**
```
--color-bg:        #FAFAFA   /* фон */
--color-primary:   #111111   /* основной текст */
--color-accent:    #C8A96E   /* золотой акцент */
--color-border:    #E5E5E5   /* разделители */
--color-muted:     #888888   /* вторичный текст */
```

**Типографика:**
- Заголовки: `Playfair Display` (serif, премиальность)
- Основной текст: `Inter` (sans-serif, читаемость)
- Размер базовый: 16px, line-height: 1.6

**Ключевые экраны:**
1. **Hero** — полноэкранное фото + слоган + CTA-кнопка
2. **Каталог** — сетка 3 колонки (desktop) / 2 колонки (mobile), фильтры сбоку
3. **Карточка товара** — галерея, описание, выбор размера, кнопка «В корзину»
4. **Корзина** — список товаров, итог, кнопка оформления
5. **Checkout** — многошаговая форма (3 шага)

## 7. Оптимизация

### 7.1 Для чего нужна

Скорость сайта напрямую влияет на конверсию: каждая секунда задержки снижает продажи на ~7%. Google учитывает Core Web Vitals при ранжировании.

### 7.2 Что будем оптимизировать

- Изображения (размер, формат)
- JavaScript bundle (code splitting)
- Шрифты (preload, display:swap)
- Рендеринг списков (виртуальный скролл)
- Кеширование (Cache-Control, Service Worker)

### 7.3 Как будем оптимизировать

```
1. Изображения → конвертация в WebP, lazy loading (<img loading="lazy">)
2. Vite → динамический import() для страниц (route-level code splitting)
3. Шрифты → <link rel="preload"> + font-display: swap
4. Pinia → пагинация / infinite scroll вместо загрузки всего каталога
5. Lighthouse CI → автоматическая проверка при каждом PR
```

### 7.4 Сравнительные результаты (план)

| Метрика | До оптимизации | После |
|---------|---------------|-------|
| LCP | ~4.2s | < 2.5s |
| FID / INP | ~180ms | < 100ms |
| CLS | 0.25 | < 0.1 |
| Lighthouse Score | ~65 | ≥ 90 |
| Bundle size | ~800kb | < 300kb |

## 10. PR-стратегия

### Позиционирование

> «MONOCHROME — для тех, кто устал от лишнего шума.»

### Каналы продвижения

**Instagram / Pinterest**
- Контент: минималистичные фото в нейтральных тонах
- Частота: 3–4 поста в неделю + stories ежедневно
- Коллаборации с микро-инфлюенсерами (10k–100k, ниша: минимализм, осознанность)

**Telegram-канал**
- «За кулисами» производства
- Анонсы новых дропов
- Эксклюзивные скидки для подписчиков

**SEO**
- Статьи в блоге: «Как составить капсульный гардероб», «Почему качество важнее количества»
- Ключевые слова: «минималистичные футболки», «премиум футболки», «базовый гардероб»

**Email-маркетинг**
- Welcome-серия (3 письма после регистрации)
- Письмо о брошенной корзине (через 1 час)
- Анонс новой коллекции
