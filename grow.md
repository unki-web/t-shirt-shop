# grow.md — Идеи для развития

## 1. Краткосрочные улучшения (1–2 месяца)

### 1.1 Технические
- **Переход на Vite + Vue SFC** — разбить монолит на компоненты, добавить Vue Router и Pinia
- **localStorage для корзины** — корзина сохраняется между сессиями
- **Поиск по каталогу** — поиск по названию и описанию товара в реальном времени
- **Skeleton-загрузка** — анимированные заглушки вместо пустого экрана при загрузке
- **Zoom на фото товара** — увеличение при наведении или клике

### 1.2 UX/UI
- **Сравнение товаров** — добавить до 3 товаров в сравнение
- **Размерная сетка** — popup с таблицей размеров и инструкцией по замерам
- **Breadcrumbs** — навигационная цепочка на странице товара
- **Recently viewed** — блок «Вы недавно смотрели» на главной и в каталоге

---

## 2. Среднесрочные улучшения (3–6 месяцев)

### 2.1 Функциональность
- **Личный кабинет** — регистрация, история заказов, адреса доставки
- **Система отзывов** — рейтинг и отзывы с фото от покупателей
- **Wishlist** — сохранение избранных товаров с синхронизацией между устройствами
- **Промокоды и скидки** — система купонов при оформлении заказа
- **Программа лояльности** — накопительные баллы за покупки

### 2.2 Контент
- **Блог** — статьи о минимализме, капсульном гардеробе, уходе за вещами (SEO-трафик)
- **Lookbook** — фотокаталог образов с кликабельными товарами
- **История каждой вещи** — страница о производстве: откуда хлопок, где шьют

### 2.3 Интеграции
- **ЮKassa / Stripe** — реальная оплата картой
- **СБП** — оплата через Систему быстрых платежей
- **СДЭК / Почта России** — расчёт доставки и трекинг заказа в реальном времени
- **Яндекс.Метрика** — аналитика поведения пользователей

---

## 3. Долгосрочные улучшения (6–12 месяцев)

### 3.1 Продвинутые функции
- **Конструктор принта** — загрузка своего рисунка на футболку (canvas API)
- **AR-примерка** — примерить футболку через камеру (WebXR API)
- **AI-подбор размера** — рекомендация размера по параметрам тела
- **Персонализация** — рекомендации товаров на основе истории просмотров

### 3.2 Масштабирование
- **Мультиязычность** — английская версия сайта (vue-i18n)
- **Мультивалютность** — цены в USD / EUR для международных продаж
- **PWA** — Progressive Web App, возможность установить как приложение
- **Мобильное приложение** — React Native или Flutter на базе того же API

### 3.3 Бизнес
- **B2B раздел** — корпоративные заказы с брендингом (мерч для компаний)
- **Дроп-система** — анонс и продажа лимитированных коллекций с таймером
- **Коллаборации** — страницы совместных коллекций с художниками/брендами
- **Подписка** — ежеквартальная доставка новинок подписчикам

---

## 4. Технический долг

Задачи, которые нужно решить по мере роста проекта:

| Задача | Когда нужно |
|--------|-------------|
| Разбить на микрофронтенды | При команде 5+ разработчиков |
| Добавить SSR (Nuxt.js) | При фокусе на SEO и органическом трафике |
| Написать unit-тесты | До запуска оплаты |
| Добавить E2E тесты (Playwright) | До запуска в продакшн |
| Настроить CI/CD | При регулярных релизах |
| Переход на TypeScript | При масштабировании команды |
